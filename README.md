# Cotxox
Vamos a construir un **prototipo** de la aplicación **Cotxox** que permite gestionar el traslado de turistas desde el aeropuerto a determinadas zonas de la isla de Mallorca.

Gracias al **main** de la aplicación podremos *ver los conductores disponibles, el precio del viaje y valorar al conductor.*



## Qué utilizamos en este proyecto?

1. For / If Else
2. ArrayList (get, size)
3. TDD (Casos Test)
4. Abstracción
5. Polimorfismo
6. Encapsulación
7. Libreria Random
